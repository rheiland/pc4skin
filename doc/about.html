GUI Overview

  • Config Basics tab: input parameters common to all models (e.g., domain grid, simulation time, choice/frequency of outputs)
  • Microenvironment tab:   microenvironment parameters (for substrates and signals)
  • Cell Types tab:          parameters that are relevant to different cell types. (Note the subtabs used to define the phenotype and custom data parameter values)
  • User Params tab:        user parameters
  • Run tab:           run the simulation (see terminal output)
  • Plot tab:           display output results (enabled after a Run)
  • Legend tab:      color legend for cells (enabled after a Run)
Clicking the 'Run Simulation' button in the Run tab will use the specified parameters and start a simulation. When a simulation begins, the Plot tab becomes enabled where output results can be visualized.

About the software

This model and cloud-hosted demo are part of the education and outreach activities for the HuBMAP project (NIH Common Fund, OT2-OD026671-01), the IU Engineered nanoBIO Node (National Science Foundation, 720625), and a cancer systems biology grant (National Cancer Institute, U01CA232137). The models are built using PhysiCell: a C++ framework for multicellular systems biology [1].

  1. A. Ghaffarizadeh, R. Heiland, S.H. Friedman, S.M. Mumenthaler, and P. Macklin. PhysiCell: an open source physics-based cell simulator for 3-D multicellular systems. PLoS Comput. Biol. 14(2):e1005991, 2018. DOI: 10.1371/journal.pcbi.1005991.
